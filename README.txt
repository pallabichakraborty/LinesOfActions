To run the code:
 Run com.loa.PlayLOA class
 
 To change Board Size:
Change the parameter BOARD_SIZE in the class com.loa.engine.board.BoardUtils
  
To setup Computer to act as a player:
Go to menu Setup->Setup Game

To change the difficulty level if Computer is one of the players:
Change the Difficulty level in menu Setup->Setup Game

To select a piece -> Left Click on it
To unselect a piece -> Right Click on it

To move a piece -> After selecting the piece, left click on the destination tile
